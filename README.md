# 目录结构
- create_data: 创建数据 Python 脚本
- data: 生成的原始数据
- pipelines: ZenML Pipeline Python 脚本
- steps: ZenML Step Python 脚本
- configs: ZenML Config Python 脚本
- run.py: ZenML Pipeline 运行脚本
- models: 生成的 ZenML Model
- test: 测试脚本
- docs: 一些文档



```bash
# Python Version 3.12
pip install -r requirements.txt
python run.py
```
